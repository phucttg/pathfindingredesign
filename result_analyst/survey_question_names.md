# Survey Question Names

## Pre/Post Repeated Questions

1. I felt confident about what the algorithm was doing while it was running.
2. I understand the underlying logic and mathematical rules (e.g., heuristics, weights) that dictate this algorithm's choices.
3. I have a clear mental model of how this specific algorithm searches through a grid or graph.
4. I can identify what makes this algorithm different from other search algorithms (e.g., heuristic-guided vs cost-based).
5. I can easily identify the most important variables this algorithm prioritizes to make its decisions.
6. I am confident that I could explain how this algorithm works to someone else.

## Demographic / Context Questions

1. Consent to participate (please tick all statements to continue)
2. Which best describes you?
3. Prior familiarity with pathfinding algorithms
4. How often do you use visual learning tools for algorithms?

## Preference Questions

1. Overall, which learning method was easier for you to follow when learning how a pathfinding algorithm works?
2. If a complete beginner wanted to learn pathfinding, which learning method would you recommend first?
3. If you had only 15-30 minutes to learn a new pathfinding algorithm, which learning method would help you most?

## Post-Use Evaluation Questions

1. This visualiser's features meet my learning needs.
2. I could use this visualiser successfully without much instruction.
3. This visualiser is easy to use.
4. This visualiser helped me understand the algorithm more effectively than theory alone.
